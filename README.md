# Market Basket Analysis  
A data mining project that applies FP-Growth for frequent itemset mining and association rule generation to uncover product associations and improve business strategies.

## Overview  
The project aims to analyze transactional data to identify patterns in customer purchasing behavior. These insights help optimize cross-selling strategies, improve product bundling, and enhance customer recommendations.

## Dataset  
The dataset contains retail transactions with features such as:  
- **BillNo**: Unique identifier for each transaction  
- **Itemname**: Description of purchased items  
- **Date**: Date of purchase  
- **CustomerID**: Anonymized customer identifier  

## Methodology  
1. **Exploratory Data Analysis (EDA)**: Understanding data structure and characteristics.  
2. **Frequent Itemset Mining**: Using FP-Growth to find frequent itemsets.  
3. **Association Rule Generation**: Discovering relationships between items in transactions.  

## Key Technologies  
- Python (Pandas, mlxtend)  
- Frequent Pattern Mining: FP-Growth Algorithm  
- Visualization: Matplotlib, Seaborn  

## Business Insights  
- Identify product combinations frequently purchased together.  
- Develop cross-selling strategies and personalized recommendations.
